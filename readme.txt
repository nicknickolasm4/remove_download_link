Remove campos de download do WooCommerce.

== Description ==

Este plugin Remove campos de download do WooCommerce.

Nenhuma configuração é necessária.

= Por que usar esse plugin? =

Esse plugin funciona para remover os links de download quando o site não utiliza esses metodos de entrega.
